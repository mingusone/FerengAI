Data Collection

Stock market data since 1980s or 1990s (CSV, API, whatever) for the big Indexes (Dow Jones, and S&P 500). Initially we will be using d daily data, but hourly can be something we use later.

Data Scaling

This data must then be compressed into a format that the neural network can process. This means all data should be turned into some number between 0 to 1. For example, if we are look at data from 9/3/2015 and the price was 100, we will find the maximum and minimum price from the last 365 days to determine the range of the scale.
